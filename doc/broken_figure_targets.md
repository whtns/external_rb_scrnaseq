## fig_1q_integrated_v2 — missing artifact

`plot_fig_04_afterall()` (numbatHelpers `figure_functions.R:16`) reads:

```
output/seurat/integrated_1q_16q/integrated_seu_1q_afterall.rds
  -> output/seurat/integrated_seu_1q_afterall.rds   # BROKEN SYMLINK (Jun 17)
```

The symlink target **does not exist anywhere on disk**, and **no pipeline target
produces it**. It was a manually-built batch-corrected 1q integration that has since
been deleted. Consequences:

- The figure cannot be rebuilt by any `tar_make()` invocation.
- To restore it, the integrated object must be regenerated by whatever ad-hoc script
  originally created it (not in this repo), then the symlink re-pointed.
